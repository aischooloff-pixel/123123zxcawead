# Telegram Media Downloader Bot

Клон бота из примера: отправляешь ссылку на Instagram / TikTok / YouTube / Pinterest, бот скачивает медиа и отправляет обратно.

## Возможности
- `/start` с приветственным сообщением и кнопкой «Добавить в группу».
- Принимает ссылку обычным сообщением.
- Скачивает медиа через `yt-dlp`.
- Отправляет фото как фото, остальные файлы как видео.
- Защита от слишком больших файлов для Telegram Bot API.

## Быстрый старт
1. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен.
2. Установи зависимости:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Запусти:
   ```bash
   export TELEGRAM_BOT_TOKEN="<YOUR_TOKEN>"
   python bot.py
   ```

## Важно
- Некоторые платформы могут ограничивать скачивание в зависимости от приватности/региона.
- Лимит на отправку файлов ботом ограничен (в коде выставлено ~49MB).
